# Raytracer

Um raytracer simples implementado em C usando raylib para renderização. Possui rastreamento básico de esferas com iluminação difusa e especular.

Desenvolvido durante este [vídeo no YouTube](https://www.youtube.com/watch?v=2ojN-5kAw3s).

## Dependências

- CMake 3.20 ou superior
- Compilador C (GCC, Clang ou MSVC)
- Git (para baixar a raylib)

O projeto baixa automaticamente a raylib 5.5 durante o processo de compilação.

## Compilação

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Executar

```bash
./Raytracer
```
