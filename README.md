# Raytracer

A simple CPU-based raytracer implementation in C using raylib for rendering. Features basic sphere raytracing with diffuse and specular lighting.

Developed during this [YouTube video](https://www.youtube.com/watch?v=2ojN-5kAw3s).

## Dependencies

- CMake 3.20 or higher
- C compiler (GCC, Clang, or MSVC)
- Git (for fetching raylib)

The project automatically downloads raylib 5.5 during the build process.

## Building

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Running

```bash
./Raytracer
```
